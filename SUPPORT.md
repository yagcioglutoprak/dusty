# Support

Dusty is a small open-source macOS cleaner, so the fastest path is to send each
question to the public place where it can help the next person too.

## Start here

- Scan results, cleanup stories, or general feedback:
  [share them in Discussions](https://github.com/yagcioglutoprak/dusty/discussions/5).
- Missing cleanup targets:
  [comment on the target request issue](https://github.com/yagcioglutoprak/dusty/issues/1)
  or open a new issue with the cleanup target template.
- Reproducible bugs:
  open a bug report with your macOS version, Dusty version, cleanup level, and
  steps to reproduce.
- Build questions or pull requests:
  read [CONTRIBUTING.md](CONTRIBUTING.md).
- Private safety or security concerns:
  follow [.github/SECURITY.md](.github/SECURITY.md) and use GitHub's private
  vulnerability reporting flow.

## What to include

For cleanup-target requests, include the path, the app or tool that creates it,
whether it regenerates safely, and any docs that prove it is cache-like data.

For bug reports, include what you expected, what happened, whether this was a
dry run or deletion run, and any relevant local log entry. Do not paste secrets,
personal file contents, or private paths you do not want public.
